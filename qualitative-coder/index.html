First-time setup

  1. Go to script.google.com → New project
  2. Paste the contents of apps-script.gs (in this folder) into the editor
  3. Click Deploy → New deployment → Web app
  4. Set Execute as: Me  ·  Who has access: Anyone
  5. Click Deploy, copy the Web App URL, and paste it above
